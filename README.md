# Pristech • Tutoriais (Modelo Visual)

Arquivos prontos para publicar no GitHub Pages.

## Como usar
1. Suba esta pasta inteira para um repositório (ex.: `pristech-tutoriais`).
2. Em **Settings → Pages**, escolha **Deploy from a branch**, **Branch: main**, **Folder: /**.
3. Acesse: `https://SEU_USUARIO.github.io/pristech-tutoriais/`.

## Onde trocar links
- Botão flutuante do Instagram: edite o link em `index.html` (procurar por `instagram.com/pristechsoftware`).

## Onde colocar seus vídeos reais
- Substitua os botões “Assistir/Baixar” pelos links dos arquivos ou players (YouTube, Vimeo, etc.).
- Substitua as imagens em `assets/posters/*.png` por thumbnails dos seus vídeos.
